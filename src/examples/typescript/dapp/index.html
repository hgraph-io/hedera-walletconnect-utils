Error Handling
Here you can reproduce some errors.
Pass any transactions with valid addresses before using this block.

INVALID_NODE_ACCOUNT ERROR

If you will provide Node Account that does't match the node account of the node the transaction was submitted to you will get this error.
To reproduce this error press the button.

TRANSACTION_EXPIRED ERROR

If you will pass transaction that has TransactionValidStart + transactionValidDuration less than current consensus time, you will get this error.
For it's reproduction pass needed information below, press button Send expired Transaction below. Wait for 15 seconds to see the result.

USER_REJECTED_METHODS ERROR

To reproduce this error just send any transaction or query to the wallet, and decline it at wallet side.
